# Web-Based Exeat Management System

A web-based student exeat management system that automates student movement requests, multi-level approvals, status tracking, and administrative monitoring within an institutional environment.

## Features
- Student exeat request submission
- Multi-level approval workflow
- Real-time status tracking
- Role-based access control (Student, Admin, Security)
- Audit logging for accountability

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas

## Project Structure
- `/src` - Frontend code
- `/server` - Backend API

## Purpose
Final-year academic project demonstrating full-stack development and workflow automation in institutional systems.
